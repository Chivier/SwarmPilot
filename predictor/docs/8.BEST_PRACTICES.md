## Training Best Practices

### 1. Collect Sufficient Training Data

**Minimum Requirements**:
- **Absolute minimum**: 10 samples (service requirement)
- **Development**: 50+ samples
- **Production**: 100+ samples for reliable predictions

**Why more is better**:
```python
# With 10 samples
model_accuracy = "Poor - high variance"

# With 50 samples
model_accuracy = "Acceptable - moderate variance"

# With 100+ samples
model_accuracy = "Good - low variance"
```

**Recommendation**:
```python
def should_train_model(sample_count):
    if sample_count < 50:
        return False, "Collect more samples"
    elif sample_count < 100:
        return True, "Acceptable for development"
    else:
        return True, "Good for production"
```

### 2. Ensure Data Quality

**Cover the Full Range**:
```python
# Bad - narrow range
training_data = [
    {"size": 1000, "runtime_ms": 100},
    {"size": 1100, "runtime_ms": 110},
    {"size": 1200, "runtime_ms": 120}
]

# Good - wide range
training_data = [
    {"size": 100, "runtime_ms": 10},
    {"size": 1000, "runtime_ms": 100},
    {"size": 10000, "runtime_ms": 1000}
]
```

**Include Edge Cases**:
```python
training_data = [
    # Small inputs
    {"size": 10, "runtime_ms": 5},
    # Typical inputs
    {"size": 1000, "runtime_ms": 100},
    # Large inputs
    {"size": 100000, "runtime_ms": 10000},
    # Unusual combinations
    {"size": 50, "batch": 256, "runtime_ms": 25}
]
```

### 3. Choose the Right Prediction Type

**Use `expect_error` when**:
- Implementing basic scheduling
- Average runtime is sufficient
- Training data is limited (<100 samples)

**Use `quantile` when**:
- You have SLA requirements
- Tail latencies matter
- You have ample training data (100+ samples)

### 4. Feature Engineering

**Keep Features Consistent**:
```python
# Define feature schema once
FEATURE_SCHEMA = {
    "image_width": int,
    "image_height": int,
    "batch_size": int,
    "model_complexity": float
}

# Use everywhere
def extract_features(task):
    return {
        "image_width": task.width,
        "image_height": task.height,
        "batch_size": task.batch,
        "model_complexity": task.complexity
    }
```

**Use Meaningful Features**:
```python
# Bad - features don't affect runtime
features = {"task_id": 123, "user_name": "alice"}

# Good - features correlate with runtime
features = {
    "input_size": 1920 * 1080,  # Affects runtime
    "batch_size": 32,             # Affects runtime
    "compression_level": 9        # Affects runtime
}
```

**Normalize Large Values**:
```python
# For very large values, consider normalization
features = {
    "image_pixels": image_width * image_height,  # Could be millions
    "batch_size": batch_size                     # Usually < 1000
}

# Better: normalize
features = {
    "image_megapixels": (image_width * image_height) / 1_000_000,
    "batch_size": batch_size
}
```

### 5. Incremental Training

**Retrain Periodically**:
```python
def should_retrain(model_info, new_samples):
    # Retrain if significant new data
    if new_samples > model_info["samples_count"] * 0.2:
        return True

    # Retrain if model is old
    days_since_training = (now() - model_info["last_trained"]).days
    if days_since_training > 30:
        return True

    return False
```

**Combine Old and New Data**:
```python
# Keep history when retraining
all_samples = old_training_data + new_training_data
train_model(model_id, platform, all_samples)
```

## Prediction Best Practices

### 1. Handle Missing Models Gracefully

```python
def predict_with_fallback(model_id, platform, features):
    try:
        response = requests.post(f"{url}/predict", json={
            "model_id": model_id,
            "platform_info": platform,
            "features": features
        })
        response.raise_for_status()
        return response.json()
    except requests.exceptions.HTTPError as e:
        if e.response.status_code == 404:
            # Model not found - use fallback
            return use_default_estimate(features)
        raise
```

### 2. Cache Predictions for Identical Requests

```python
import hashlib
import json

prediction_cache = {}

def get_cached_prediction(model_id, platform, features):
    # Create cache key
    key_data = json.dumps({
        "model_id": model_id,
        "platform": platform,
        "features": features
    }, sort_keys=True)
    cache_key = hashlib.md5(key_data.encode()).hexdigest()

    # Check cache
    if cache_key in prediction_cache:
        return prediction_cache[cache_key]

    # Get prediction
    prediction = get_prediction(model_id, platform, features)
    prediction_cache[cache_key] = prediction
    return prediction
```

### 3. Use Appropriate Timeouts

```python
import requests

# Short timeout for predictions (service should be fast)
response = requests.post(
    f"{url}/predict",
    json=data,
    timeout=2.0  # 2 seconds
)

# Longer timeout for training (can be slow)
response = requests.post(
    f"{url}/train",
    json=data,
    timeout=30.0  # 30 seconds

```

### 4. Validate Features Before Prediction

```python
def validate_features(features, expected_schema):
    """Validate features match expected schema"""
    # Check all required features present
    missing = set(expected_schema.keys()) - set(features.keys())
    if missing:
        raise ValueError(f"Missing features: {missing}")

    # Check types
    for key, expected_type in expected_schema.items():
        if not isinstance(features[key], expected_type):
            raise TypeError(
                f"Feature '{key}' has wrong type: "
                f"expected {expected_type}, got {type(features[key])}"
            )

    return True

# Usage
SCHEMA = {"image_width": int, "image_height": int, "batch_size": int}
validate_features(task_features, SCHEMA)
prediction = get_prediction(model_id, platform, task_features)
```

## Performance Optimization

### 1. Batch Predictions

```python
# Bad - sequential predictions
predictions = []
for task in tasks:
    pred = get_prediction(model_id, platform, task.features)
    predictions.append(pred)

# Better - concurrent predictions
import concurrent.futures

def predict_task(task):
    return get_prediction(model_id, platform, task.features)

with concurrent.futures.ThreadPoolExecutor(max_workers=10) as executor:
    predictions = list(executor.map(predict_task, tasks))
```

### 2. Minimize Feature Computation

```python
# Bad - compute features every time
for task in tasks:
    features = compute_expensive_features(task)  # Slow!
    prediction = get_prediction(model_id, platform, features)

# Good - compute once, cache
feature_cache = {}
for task in tasks:
    if task.id not in feature_cache:
        feature_cache[task.id] = compute_expensive_features(task)
    prediction = get_prediction(model_id, platform, feature_cache[task.id])
```

### 3. Use Experiment Mode for Testing

```python
# During development/testing, use experiment mode
if is_development():
    features["exp_runtime"] = estimated_runtime
    # Fast, no model required

# In production, use trained models
prediction = get_prediction(model_id, platform, features)
```

```python
def get_prediction_with_fallback(model_id, platform, features):
    """Get prediction with multiple fallback strategies"""
    # Try primary predictor
```

### 3. Load Testing

```python
import time
import statistics

def load_test(num_requests=1000):
    """Test predictor under load"""
    latencies = []

    for i in range(num_requests):
        start = time.time()
        try:
            response = requests.post(f"{url}/predict", json=test_data)
```

### 4. Error Handling

```python
from requests.adapters import HTTPAdapter
from urllib3.util.retry import Retry

def create_robust_session():
    """Create session with retry logic"""
    session = requests.Session()

    retry_strategy = Retry(
        total=3,
        backoff_factor=1,
        status_forcelist=[429, 500, 502, 503, 504],
        allowed_methods=["POST", "GET"]
    )

    adapter = HTTPAdapter(max_retries=retry_strategy)
    session.mount("http://", adapter)
    session.mount("https://", adapter)

    return session

# Usage
session = create_robust_session()
```

## Monitoring and Maintenance

### 1. Track Prediction Accuracy

```python
class PredictionMonitor:
    def __init__(self):
        self.predictions = []
        self.actuals = []

    def record_prediction(self, predicted_ms):
        self.predictions.append(predicted_ms)

    def record_actual(self, actual_ms):
        self.actuals.append(actual_ms)

    def get_accuracy(self):
        """Calculate prediction accuracy"""
        errors = [
            abs(pred - actual) / actual
            for pred, actual in zip(self.predictions, self.actuals)
        ]
        return {
            "mean_error": statistics.mean(errors),
            "median_error": statistics.median(errors),
            "p95_error": statistics.quantiles(errors, n=20)[18]
        }

    def should_retrain(self):
        """Check if model needs retraining"""
        accuracy = self.get_accuracy()
        return accuracy["mean_error"] > 0.20  # 20% error threshold
```

### 2. Log Important Metrics

```python
import logging

logger = logging.getLogger(__name__)

def predict_with_logging(model_id, platform, features):

```

```python
def maintenance_loop():
```

```python
def check_and_alert():
```
