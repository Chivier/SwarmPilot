# Quick Start Guide

Get up and running with the Predictor service in 5 minutes.

## Prerequisites

- Predictor service running on `http://localhost:8000` (or your configured URL)
- `curl` or any HTTP client
- Basic understanding of HTTP/JSON

## Step 1: Verify Service is Running

Check that the service is healthy:

```bash
curl http://localhost:8000/health
```

**Expected response**:
```json
{
  "status": "healthy"
}
```

If you get a connection error, ensure the service is started and accessible.

## Step 2: Train Your First Model

Train a simple model with sample data:

```bash
curl -X POST http://localhost:8000/train \
  -H "Content-Type: application/json" \
  -d '{
    "model_id": "quickstart-model",
    "platform_info": {
      "software_name": "pytorch",
      "software_version": "2.0.1",
      "hardware_name": "nvidia-a100"
    },
    "prediction_type": "expect_error",
    "features_list": [
      {"input_size": 512, "batch_size": 16, "runtime_ms": 25.3},
      {"input_size": 512, "batch_size": 32, "runtime_ms": 48.7},
      {"input_size": 1024, "batch_size": 16, "runtime_ms": 45.2},
      {"input_size": 1024, "batch_size": 32, "runtime_ms": 89.5},
      {"input_size": 2048, "batch_size": 16, "runtime_ms": 89.1},
      {"input_size": 2048, "batch_size": 32, "runtime_ms": 172.3},
      {"input_size": 4096, "batch_size": 16, "runtime_ms": 178.7},
      {"input_size": 4096, "batch_size": 32, "runtime_ms": 342.1},
      {"input_size": 512, "batch_size": 64, "runtime_ms": 95.4},
      {"input_size": 1024, "batch_size": 64, "runtime_ms": 175.8}
    ]
  }'
```

**Expected response**:
```json
{
  "status": "success",
  "message": "Model trained successfully",
  "model_key": "quickstart-model__pytorch-2.0.1__nvidia-a100",
  "samples_trained": 10
}
```

## Step 3: Make a Prediction

Get a runtime prediction for new input:

```bash
curl -X POST http://localhost:8000/predict \
  -H "Content-Type: application/json" \
  -d '{
    "model_id": "quickstart-model",
    "platform_info": {
      "software_name": "pytorch",
      "software_version": "2.0.1",
      "hardware_name": "nvidia-a100"
    },
    "prediction_type": "expect_error",
    "features": {
      "input_size": 1536,
      "batch_size": 24
    }
  }'
```

**Expected response**:
```json
{
  "model_id": "quickstart-model",
  "platform_info": {
    "software_name": "pytorch",
    "software_version": "2.0.1",
    "hardware_name": "nvidia-a100"
  },
  "prediction_type": "expect_error",
  "result": {
    "expected_runtime_ms": 110.5,
    "error_margin_ms": 12.3
  }
}
```

**Interpretation**: The task will likely take around 110.5ms ± 12.3ms to complete.

## Step 4: List Available Models

View all trained models:

```bash
curl http://localhost:8000/list
```

**Expected response**:
```json
{
  "models": [
    {
      "model_id": "quickstart-model",
      "platform_info": {
        "software_name": "pytorch",
        "software_version": "2.0.1",
        "hardware_name": "nvidia-a100"
      },
      "prediction_type": "expect_error",
      "samples_count": 10,
      "last_trained": "2025-10-31T12:34:56Z"
    }
  ]
}
```

## Common Issues

### Issue: Connection Refused

**Problem**: `curl: (7) Failed to connect to localhost port 8000`

**Solution**:
- Check if the service is running
- Verify the correct port (default is 8000)
- Check firewall settings

### Issue: Model Not Found (404)

**Problem**:
```json
{
  "error": "Model not found",
  "model_id": "quickstart-model"
}
```

**Solution**:
- Verify you trained the model first (Step 2)
- Check that `model_id` and `platform_info` match exactly
- Use `/list` endpoint to see available models

### Issue: Invalid Features (400)

**Problem**:
```json
{
  "error": "Invalid features",
  "message": "Missing required feature: 'batch_size'"
}
```

**Solution**:
- Ensure all features used during training are present
- Check feature names match exactly (case-sensitive)
- Verify feature types (numbers vs strings)

## Using with Python

Quick Python example:

```python
import requests

PREDICTOR_URL = "http://localhost:8000"

# Train a model
training_request = {
    "model_id": "python-model",
    "platform_info": {
        "software_name": "pytorch",
        "software_version": "2.0.1",
        "hardware_name": "nvidia-a100"
    },
    "prediction_type": "expect_error",
    "features_list": [
        {"size": 1024, "batch": 16, "runtime_ms": 45.2},
        {"size": 2048, "batch": 16, "runtime_ms": 89.5},
        {"size": 1024, "batch": 32, "runtime_ms": 42.1},
        {"size": 2048, "batch": 32, "runtime_ms": 85.3},
        {"size": 4096, "batch": 16, "runtime_ms": 178.7},
        {"size": 4096, "batch": 32, "runtime_ms": 172.4},
        {"size": 512, "batch": 16, "runtime_ms": 22.8},
        {"size": 512, "batch": 32, "runtime_ms": 21.5},
        {"size": 8192, "batch": 16, "runtime_ms": 356.2},
        {"size": 8192, "batch": 32, "runtime_ms": 348.9}
    ]
}

response = requests.post(f"{PREDICTOR_URL}/train", json=training_request)
print("Training result:", response.json())

# Get prediction
prediction_request = {
    "model_id": "python-model",
    "platform_info": {
        "software_name": "pytorch",
        "software_version": "2.0.1",
        "hardware_name": "nvidia-a100"
    },
    "prediction_type": "expect_error",
    "features": {
        "size": 3072,
        "batch": 24
    }
}

response = requests.post(f"{PREDICTOR_URL}/predict", json=prediction_request)
result = response.json()
print(f"Predicted runtime: {result['result']['expected_runtime_ms']:.2f}ms")
```

## Summary

**Basic workflow**:
1. Check service health (`/health`)
2. Train a model (`/train`)
3. Make predictions (`/predict`)
4. List models (`/list`)

**Key concepts**:
- **Model ID**: Unique identifier for your model
- **Platform Info**: Hardware/software environment
- **Features**: Input parameters affecting runtime
- **Prediction Type**: `expect_error` (simple) or `quantile` (detailed)

**Remember**:
- Train models with 50+ samples for good accuracy
- Feature names must match exactly between training and prediction
- Use experiment mode for testing without training data

Happy predicting!
